# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the repository](https://github.com/changesets/changesets)

Each changeset is stored in a markdown file under this directory. When you run `pnpm changeset`, a new
file is created describing which packages changed and what kind of semver bump they need.

When ready to release, run `pnpm version-packages` to consume all changesets and bump versions, then
`pnpm release` to build and publish.
